# 港股新股监测系统 🚀📊 v2.0

基于AI的智能港股新股分析与预警平台

## 项目介绍

港股新股监测系统是一个专注于港股IPO（首次公开募股）信息展示的数据平台，帮助投资者实时掌握港股新股动态，包括招股中、即将上市、已上市新股的表现数据。

## 主要功能

### 📋 招股中新股
- 实时展示正在招股的新股信息
- 招股价格、每手股数、入场费
- 超额认购倍数（每日更新）
- 多维度评分分析
- 投资亮点与风险因素
- 专业投资建议

### 🚀 明日上市
- 即将上市新股预览
- 发行价、暗盘表现
- 超额认购数据
- 公司亮点分析

### 📈 已上市新股表现
- 2026年新股完整列表
- 暗盘涨跌幅、首日涨跌幅、累计涨幅
- 按行业分类筛选

### 📊 行业估值对比
- AI芯片/算力、创新药、智能硬件ODM、商业物联网
- PE、PB估值分析
- 投资建议
